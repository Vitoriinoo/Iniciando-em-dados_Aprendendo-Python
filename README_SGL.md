## Sistema de Gestão de Loja
Este repositório contém um sistema de automação de vendas desenvolvido em Python. O projeto simula o fluxo de decisão de uma loja virtual, realizando a validação de dados de clientes, cálculos dinâmicos de descontos cumulativos e logística de frete.

O objetivo principal deste código é demonstrar a aplicação prática de lógica de programação para resolver problemas de negócios. A solução garante que múltiplas variáveis comerciais operem em conjunto e de forma hierárquica, resultando na geração de um relatório final estruturado.

## Competências Desenvolvidas
Domínio de estruturas condicionais simples e aninhadas (if, elif, else).

Implementação de regras de negócio complexas com travas de segurança lógica.

Utilização de operadores lógicos e de comparação encadeada.

Tratamento e formatação de strings para interface de relatórios no console.

Aplicação de boas práticas de programação, priorizando código limpo e legibilidade.
